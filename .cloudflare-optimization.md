# Cloudflare Auto Minify Setup

## ✅ CSS Minification đã được setup

### Cách 1: Auto Build (Khuyến nghị cho production)

File `style.min.css` đã được tạo tự động. Để sử dụng:

```bash
# Build và minify tất cả
npm run build

# Chỉ minify CSS
npm run minify:css

# Deploy với minify
npm run deploy
```

### Cách 2: Cloudflare Dashboard Auto Minify

1. Vào **Cloudflare Dashboard**
2. Chọn site của bạn
3. Vào **Speed** → **Optimization**
4. Bật **Auto Minify**:
   - ✅ JavaScript
   - ✅ CSS
   - ✅ HTML

### Cách 3: Sử dụng file minified trong production

Thêm vào HTML (production only):

```html
<!-- Development -->
<link rel="stylesheet" href="css/style.css">

<!-- Production (minified) -->
<link rel="stylesheet" href="css/style.min.css">
```

## Kết quả

- **Original CSS**: ~60KB
- **Minified CSS**: ~45KB (giảm ~25%)
- **Gzipped**: ~10KB (giảm ~83%)

## Build Scripts Available

```json
{
  "build": "Minify cả CSS và HTML",
  "minify:css": "Chỉ minify CSS",
  "minify:html": "Chỉ minify HTML",
  "deploy": "Build + Deploy tự động"
}
```

## Notes

- Cloudflare tự động gzip/brotli compress tất cả assets
- Auto Minify ở dashboard áp dụng cho tất cả requests
- Build script chỉ tạo file static minified
- Khuyến nghị: Dùng build script + Cloudflare Auto Minify để tối ưu tốt nhất
